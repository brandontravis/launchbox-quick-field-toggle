# QuickFieldToggle

Full documentation, examples, and configuration guide:  
**https://github.com/brandontravis/launchbox-quick-field-toggle**

## Quick Start

1. Copy this folder to `LaunchBox\Plugins\`
2. Rename `quickfieldtoggle.sample-simple.json` to `quickfieldtoggle.json`
3. Restart LaunchBox
4. Right-click any game → enjoy your new menu options

> If the plugin doesn't load, right-click `QuickFieldToggle.dll` → Properties → Unblock
